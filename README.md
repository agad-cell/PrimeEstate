**Prime Estate** is a real estate website or more likely a webpage that is built to show the elegant properties and how real webpages work. 
It is a static webpage with some animations and effects.
# My Story and Motivation
Three things motivated me to build this webpage one to **learn responsiveness** and **how real webpages are designed and structured** and third my **interest toward real estate business**...

# what I learnt
After this webpage I know enough about responsiveness and UI design, i know about the screen sizes and how do sections work and how small icons are used....

# What this webpage does
This webpage showcases property listings and the services provided by real estate business, shows the modern animations, glowing effects and mobile-friendly navigation......

Overall the webpages includes property listinsg, services provided, a header, a footer and some other sections of real estate websites....
